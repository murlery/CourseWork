Описание работы с программой "Эвелирный магазин"
1. Описание:
   Эта программа предназначена для управления записями о ювелирных изделиях в магазине. 
   С помощью нее вы можете добавлять, редактировать и удалять информацию о изделиях, а также проводить поиск и сортировку данных.

2. Установка и запуск:
   Программа не требует установки. Просто запустите исполняемый файл для начала работы.

3. Функции и возможности:
   Файл:
     - Создать: создает новый файл для ввода данных о зверях и птицах.
   - Открыть: позволяет выбрать и открыть существующий файл с данными.
   - Сохранить: сохраняет текущие данные в открытый файл.
   - Сохранить как: позволяет сохранить текущие данные в новый файл.
   Правка:
   - Добавление записи. Введите информацию в поля ввода и нажмите "Добавить".
   - Редактирование записи: Выберите строку в таблице, нажмите "Редактировать", измените информацию и нажмите "Сохранить".
   - Удаление записи: Выберите строку в таблице и нажмите "Удалить".
   Обработка:
   -Поиск: Введите название изделия в поле поиска и нажмите "Поиск".
   - Сортировка: Вы можете отсортировать данные по цене (возрастание, убывание) или по названию.

4. Ввод данных:
   В каждую запись вводятся следующие данные: название изделия, цена, вес изделия, изготовитель, основной материал, драгоценный камень, вес камня (в каратах).
